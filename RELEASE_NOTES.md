# Car Talk v0.1.0

Initial release of Car Talk — anonymous inter-vehicle messaging for Android.

## What's New
- 🚗 Register up to 3 vehicles (license plate, make, model, color)
- 🔍 Search any driver by license plate
- 💬 Send anonymous messages (200 characters)
- 👥 Friend system with 500-character friend messages
- 📲 Push notifications with Android Auto support
- 🔒 30-day message retention, 5 anonymous messages/day cap
- 🛡️ Built-in profanity filter

## Installation
Download the APK from the Assets below and install on Android 14+.

> Enable "Install from unknown sources" if installing outside the Play Store.
